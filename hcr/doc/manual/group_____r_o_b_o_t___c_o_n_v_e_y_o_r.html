◆ clink_robot_conveyor_offset_calculate()

CLINK_EXPORT CLINK_API_RESULT clink_robot_conveyor_offset_calculate ( uint32_t  robot_id,
clink_float_t  velocity,
clink_float_t  acceleration,
clink_float_t *  p_offset 
)

conveyor 모션에서 처음 가속 구간동안 생기는 거리 오프셋 값을 계산한다.

Parameters
[in]robot_idclink_robot_create() 를 통해 반환받은 로봇 인스턴스 ID
[in]velocityconveyor의 속도 (로봇의 속도) [mm/s]
[in]acceleration로봇의 가속도 [mm/s^2]
[out]p_offset계산된 오프셋 값 [mm]
Remarks
conveyor 속도는 일정한대 비해, 로봇은 첫 모션에서 가속구간이 있으므로
해당 구간에서 뒤 떨어지는 거리를 계산하여 미리 반영하기 위함
Returns
returnable error code

[out]p_offset계산된 오프셋 값 [mm]
