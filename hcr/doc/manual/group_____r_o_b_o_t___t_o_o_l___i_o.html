CLINK_EXPORT CLINK_API_RESULT clink_robot_tool_io_power_voltage_set ( uint32_t  robot_id,
CLINK_TOOL_IO_POWER_VOLTAGE_OPTION  voltage_option,
char_t  b_sync 
)

Tool I/O의 Power 전압을 셋팅한다.

Parameters
[in]robot_idclink_robot_create() 를 통해 반환받은 로봇 인스턴스 ID
[in]voltage_optionCLINK_TOOL_IO_POWER_VOLTAGE_OPTION 참조:
CLINK_TOOL_POWER_VOLTAGE_OPTION_0,
CLINK_TOOL_POWER_VOLTAGE_OPTION_12,
CLINK_TOOL_POWER_VOLTAGE_OPTION_24 입력가능
[in]b_sync함수의 synchronous 동작 유무를 선택.
0 인 경우 함수는 값을 셋팅하는 명령을 제어SW 에 전송한 뒤
바로 리턴. 즉, delay없음
그 외의 값인 경우 함수는 제어SW에 전송한 명령의
반영을 확인하고 리턴, 즉 100 - 200 ms delay 가능
Remarks
셋팅된 voltageOption의 voltage 옵션은 Tool Digital IO 전압의 기준이 된다.
Offline Simulator 및 simulation robot 지원불가.
Returns
returnable error code
