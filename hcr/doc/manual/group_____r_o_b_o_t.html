◆ clink_robot_emergency_stop()

CLINK_EXPORT CLINK_API_RESULT clink_robot_emergency_stop ( uint32_t  robot_id,
clink_float_t  stop_time 
)

현재 로봇을 응급정지 한다.

Parameters
[in]robot_idclink_robot_create() 를 통해 반환받은 로봇 인스턴스 ID
[in]stop_time단위 : 초 (sec)
명령 지령 시점부터 command velocity 가 0 가 될때까지의 시간.
Remarks
clink_robot_stop() 의 경우, 로봇은 joint 모션을 실행중에는
각 joint 를 멈추고, tcp 모션을 실행중에는 tcp 수준에서 정지한다.
그리고, 설정된 velocity profiler 를 활용하여 stop 한다.
clink_robot_emergency_stop() 의 경우에는 현재 실행중인
모든 모션을 무시하고, 현재 각 축의 velocity 를 설정된
stop_time 시간내에 모두 0 가 되게끔 속도를 profiling 하여
로봇을 정지시킨다.
Returns
returnable error code
