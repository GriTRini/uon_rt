CLINK_EXPORT CLINK_API_RESULT clink_robot_spec_joint_acc_max_get ( uint32_t  robot_id,
uint32_t  joint_index,
clink_float_t *  p_acc_max 
)

로봇의 specification 상의 해당 Joint 의 최대 가속도를 반환한다.

Parameters
[in]robot_idclink_robot_create() 를 통해 반환받은 로봇 인스턴스 ID
[in]joint_index해당 하는 Joint 의 index
6축 로봇의 경우, 첫 번째 축은 0
마지막 축은 5
[out]p_acc_max최대 가속도 값 (단위 : degree / second^2)
Remarks
로봇의 spec 은 사용자가 설정하는 값이 아니고
로봇의 설계상의 spec 이다. 따라서 별도로 set
함수를 제공하지 않는다.
Returns
returnable error code
