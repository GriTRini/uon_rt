입력한 joint angle 로 로봇을 이동시킨다.

Parameters
[in]robot_idclink_robot_create() 를 통해 반환받은 로봇 인스턴스 ID
[in]speed_factor구동 속도 설정 값
최소 : 0.1
최대 : 1.0
최대 (즉, 1.0) 으로 입력한 경우 로봇은
spec 상의 최대 속도로 동작한다.
[in]num_of_jointsjoint_angles 배열의 크기
로봇의 joint 개수 이상이어야 한다.
[in]joint_angles각 joint 의 모션 종료 각도
(단위 : degree)
움직이지 않고 싶은 joint 의 경우에는
현재 joint 의 각도를 그대로 입력한다.
joint_angles 배열의 size 는 로봇의 joint 개수
이상이어야 한다. 즉 6축 로봇인 경우 6개 이상의
배열 size 를 보장해야 한다.
Remarks
joint_angles 배열의 size 가 로봇 joint 의 개수
이하인 경우, segmentation fault 를 발생시킬 수 있다.
Returns
returnable error code
