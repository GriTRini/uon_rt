◆ clink_robot_ext_profiler_pos_ort_euler_add()

CLINK_EXPORT CLINK_API_RESULT clink_robot_ext_profiler_pos_ort_euler_add ( uint32_t  robot_id,
clink_float_t  pos_x,
clink_float_t  pos_y,
clink_float_t  pos_z,
clink_float_t  ort_x,
clink_float_t  ort_y,
clink_float_t  ort_z 
)

External Profiler 모드로서, 로봇에 전달할 command position 및
orientation 성분을 전송한다.
이 함수는 정해진 시간 (5 ms) 에 정확히 맞추어 호출되어야 한다.

Parameters
[in]robot_idclink_robot_create() 를 통해 반환받은 로봇 인스턴스 ID
[in]pos_xPosition 데이터의 X 성분, base 좌표계에서 TCP 위치 기준
[in]pos_yPosition 데이터의 Y 성분, base 좌표계에서 TCP 위치 기준
[in]pos_zPosition 데이터의 Z 성분, base 좌표계에서 TCP 위치 기준
[in]ort_xOrientation 의 X 성분, base 좌표계에서 TCP 기준, degree 값 (즉, radian 아님)
[in]ort_yOrientation 의 Y 성분, base 좌표계에서 TCP 기준, degree 값 (즉, radian 아님)
[in]ort_zOrientation 의 Z 성분, base 좌표계에서 TCP 기준, degree 값 (즉, radian 아님)
Remarks
같은 위치를 계속 추가할 경우 로봇 이동하지 않고,
해당 위치를 지속적으로 유지한다.
5ms 간격으로 추가되는 로봇의 위치는 속도 및 가속도가
적용되는 자체 프로파일러를 사용하여 생성할 것을 권장한다.
로봇은 각 Joint마다 속도 및 가속도 제한이 있으며(제한 위반시 stop),
모두 불연속한 부분없이 부드럽게 이어져야 한다.
3개 이상의 point 를 입력한 순간부터 모션이 시작되며,
시작 시 ADD한 모든 point는 반영된다.
(5ms 간격의 point들을 내부 제어 주기에 맞게
interpolation 하는 과정에 최소 3개 이상의 point가 필요)
Returns
returnable error code

CLINK_EXPORT CLINK_API_RESULT clink_robot_ext_move_switch_set ( uint32_t  robot_id,
CLINK_SWITCH  on_off 
)

로봇의 External Move 모드를 enable / disable 한다.

Parameters
[in]robot_idclink_robot_create() 를 통해 반환받은 로봇 인스턴스 ID
[in]on_offCLINK_SWITCH_ON : External Move 모드 ON
CLINK_SWITCH_OFF : External Move 모드 OFF
Remarks
External Move 는 주어진 target position 으로 이동한다.
사용자는 profile 등에 대한 고려 없이 원하는 위치의 target 만을
전송하면 로봇 제어기가 알아서 해당 위치로 이동하는 것이
external profiler 와 다르다.
Profile 된 데이터가 입수 가능하고 세밀한 제어를 원하는 경우 External Profiler 를 사용하고,
Path 속도 등이 중요하지 않고, 편하게 어떤 위치로 이동하고 싶은 경우에는 External Move 를 사용한다.
Returns
returnable error code
