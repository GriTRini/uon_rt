CLINK_EXPORT CLINK_API_RESULT clink_cbox_reset ( )

Control Box 와 연결된 모든 로봇의 상태를 초기화 한다.
Error 가 있는 경우, Error Clear 를 시도한다.
.

Remarks
내부적으로 Control Box 를 리셋한 후,
clink_robot_reset() 를 호출하여 로봇을 초기화한다.
Returns
returnable error code

◆ clink_cbox_master_operational_mode_set()

CLINK_EXPORT CLINK_API_RESULT clink_cbox_master_operational_mode_set ( CLINK_OPERATIONAL_MODE  operational_mode)

cbox의 Operational Mode 를 설정한다.

Parameters
[in]operational_modeEtherCAT 의 4가지 operational mode 중 한 가지를 선택한다.
Remarks
내부적으로 연결된 모든 slave의 operational mode 를 변경한다.
Servo On 을 하기 위해서는 CLINK_OPERATIONAL_MODE_OPERATIONAL
를 선택해야 한다. 그 외 모드는 특수한 경우를 제외하고는
일반 사용자가 선택할 필요는 없다.
Returns
returnable error code
