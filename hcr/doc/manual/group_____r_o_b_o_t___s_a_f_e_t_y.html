◆ clink_robot_safety_limit_switch_set()

CLINK_EXPORT CLINK_API_RESULT clink_robot_safety_limit_switch_set ( uint32_t  robot_id,
CLINK_SWITCH  on_off 
)

로봇의 안전설정의 유효화 혹은 무효화한다.

Parameters
[in]robot_idclink_robot_create() 를 통해 반환받은 로봇 인스턴스 ID
[in]on_offCLINK_SWITCH_ON : 안전설정 유효화
CLINK_SWITCH_OFF : 안전설정 무효화
Remarks
안전설정 무효화는 안전설정을 위반하여
로봇을 안전영역으로 복귀시킬 때 사용하는 것을
목적으로 한다. 일반적인 상황에서 안전설정을
무효화하면, 안전설정을 위반해도 로봇은 동작을
계속하기 때문에 위험할 수 있다.
Returns
returnable error code
