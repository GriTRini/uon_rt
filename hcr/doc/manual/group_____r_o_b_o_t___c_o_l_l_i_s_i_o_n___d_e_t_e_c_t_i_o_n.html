CLINK_SWITCH  on_off 
)

로봇의 충돌감지 기능을 on / off 한다.

Parameters
[in]robot_idclink_robot_create() 를 통해 반환받은 로봇 인스턴스 ID
[in]on_offCLINK_SWITCH_ON : 충돌감지 실행
CLINK_SWITCH_OFF : 충돌감지 실행하지 않음
Remarks
Collision Detection 의 비활성화는 협동로봇으로 로봇을 사용하지 않는다는
의미로서 시스템 담당자는 해당 로봇이 협동로봇의 안전기능인 충돌감지를
사용하지 않는다는 것을 시각적 도구를 통해 제공해야할 의무가 있음.
Returns
returnable error code
