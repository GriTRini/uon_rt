주어진 Pose (Position + Orientation) 에 해당 하는 각도를 리턴한다.
Inverse Kinematics 에서 가능한 8개의 모든 해를 반환한다.

Parameters
[in]robot_idclink_robot_create() 를 통해 반환받은 로봇 인스턴스 ID
[in]p_pos위치 X,Y,Z (크기 3이상 배열)
[in]p_ort방향 X,Y,Z (크기 3이상 배열)
[out]p_joint_angles로봇의 각 조인트 angle 반환(단위: Degree)
8개의 해가 차례대로 저장된다.
즉, 배열의 크기는 조인트 개수 X 8 만큼 할당되어야 한다.
해가 존재하지 않을 경우 해당 배열 구간의 값은 invalid 하다.
[out]p_validity8개의 해 중, 존재하는 경우 1, 존재하지 않는 경우 0을 반환한다.
해당 배열의 크기는 8만큼 할당되어야 한다.
Returns
returnable error code

Parameters

[in]start_angles시작 각도, num_of_joints 갯수 만큼의 배열 [degree]
[in]end_angles끝 각도, num_of_joints 갯수 만큼의 배열 [degree]
[in]num_of_points궤적을 이루는 point의 갯수 설정
[in]pos_type계산되는 position 값이 flange 끝 단인지 tcp 끝 단인지 결정
[out]p_x계산된 좌표의 x 성분 배열, num_of_points 만큼의 크기 할당필요 [mm]
