)

Robot TCP 를 임의의 방향으로 이동한다. (JOG 모드)
.

Parameters
[in]robot_idclink_robot_create() 를 통해 반환받은 로봇 인스턴스 ID
[in]ref_coord이동 방향의 기준 좌표계
CLINK_REF_COORDINATE_BASE 또는,
CLINK_REF_COORDINATE_TCP
[in]dir_xJOG 방향 중 X 성분
[in]dir_yJOG 방향 중 Y 성분
[in]dir_zJOG 방향 중 Z 성분
[in]speed로봇의 이동 속도 (단위 : millimeter / second)
[in]distance이동 distance (단위 : millimeter)
JOG 를 통해 이동할 거리를 지정한다.
충분히 큰 거리 (예: 100 meter 에 해당하는 100000)
를 입력하는 경우, 로봇은 workspace 의 한계점까지 이동한다.
작은 거리를 입력하는 경우, 해당 거리만을 이동한다.
Returns
returnable error code
